# Theme

The current source of truth is the uploaded Pirate OS mobile concept board and the user's written breakdown.

Visual direction:

- mobile-first portrait game UI
- clean black-market cyberdeck
- dark navy / near-black background
- restrained cyan and magenta neon accents
- circular Energy and Heat meters
- one active signal card
- one compact S1LKROAD market list
- rounded glass panels, thin glowing outlines, generous negative space
- no Matrix-green overload, no ASCII desktop shell, no busy dashboard chrome

Runtime tokens live in `src/ui/theme.ts`.
