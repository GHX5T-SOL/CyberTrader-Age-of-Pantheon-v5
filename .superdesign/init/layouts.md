# Layouts

The persistent Pirate OS layout is implemented in `src/App.tsx` as `CyberdeckShell`.

It provides:

- top OS bar with `/dev/tty-pantheon` breadcrumb
- menu shortcut
- persistent bottom chrome
- wallet chip, 0BOL, $OBOL locked state
- Energy, Heat, Rank, and OS tier

The shell wraps all post-login screens.
